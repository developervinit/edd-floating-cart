Standalone foundation for an Easy Digital Downloads floating cart plugin.

== Description ==

This release provides the plugin scaffold only.

== Installation ==

1. Upload the `edd-floating-cart` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" screen in WordPress.
